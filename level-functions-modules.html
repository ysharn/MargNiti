Functions & Modules

Learn to write reusable functions and organize code using modules.

1. What are Functions?

Functions are blocks of reusable code that perform a specific task. They help in organizing code and avoiding repetition.

2. What are Modules?

Modules are files containing Python code that can be imported and reused in other programs.

3. Key Concepts

  • Functions: Define reusable blocks of code.
  • Modules: Organize code into separate files for better maintainability.
  • Importing: Use code from other modules in your program.

4. Examples

  • Write a function to calculate the area of a circle.
  • Create a module to handle mathematical operations.
